# Investor Magnet Swarm

Use OpenAgent as the user-facing entrypoint. Route the request to the smallest useful investor magnet team.

## Instructions

1. Identify the investor objective:
   - warm inbound interest
   - funding round preparation
   - strategic investor narrative
   - PR/media momentum
   - LinkedIn category leadership
   - analyst/event credibility
   - social proof and validation
   - crisis-to-opportunity response
2. Select relevant investor agents.
3. Separate public narrative from private investor memo content.
4. Map proof, metrics, permissions, and data-room gaps.
5. Ask for approval before any external-facing claim or outreach.
