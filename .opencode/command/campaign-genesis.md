# Campaign Genesis

Build a campaign system in parallel:

1. MarketIntelligenceAgent: segment, competitors, triggers, whitespace.
2. CustomerResearchAgent: ICP, personas, pain points, objections.
3. BrandStrategyAgent: positioning and proof hierarchy.
4. ContentSwarmAgent: content matrix and asset briefs.
5. ConversionAgent: landing pages, offers, A/B tests.
6. PRCommunicationsAgent: journalist/event/thought-leadership angles.
7. PerformanceAnalyticsAgent: metrics and experiment readout plan.
8. ChiefGrowthOfficerAgent: final KPI and strategy arbitration.

Return a campaign plan and request approval before creating final assets.
